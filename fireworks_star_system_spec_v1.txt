花火ゲーム：星（Star）システム仕様書 v1.0
作成日: 2026-01-30

============================================================
0. この仕様書の前提（重要）
============================================================
- 本仕様書は「花火の見え方」をゲームで再現するための VFX/挙動仕様です。
- 現実の花火を製造・再現するためのレシピ、配合、工作手順、施工方法は扱いません。
- ここでいう「層（Layer）」は物理的な製造層ではなく、ゲーム内の“時間順に発火する見え方モジュール”
  （＝エフェクトスタック / 演出スタック）を指します。

============================================================
1. 目的 / スコープ
============================================================
- 星（Star）の「見え方」を、(1) 種類分類、(2) 部品（層）ライブラリ、(3) 組み立て表（推奨スタック）
  の3点で統一し、実装・データ駆動化（スプレッドシート/JSON化）しやすくする。

============================================================
2. 用語
============================================================
- 星（Star）:
  打ち上げ後の開花を構成する個々の発光体（ゲームではパーティクル要素/スプライト要素単位）。
- 見え方の核（Kernel）:
  「尾がある/垂れる/明滅する/割れる」など、色ではなく現象の中心となる見え方。
- 色（Color）:
  基本はパラメータ扱い。赤/青/緑…などはカテゴリ分けせず、同一モジュールに対する色差分とする。
- Prime（起動）:
  星スタックの開始トリガ（点火フラッシュ/開始スパーク等の最小演出）。実装上の“開始保証”。

【注意】Ghost（ゴースト）という語は英語圏で別概念（ゴーストシェル等）にも使われやすい。
本仕様では「消えそうで消えない青白い弱光」は Afterglow（残光）として扱う。

============================================================
3. 星の分類（色違いは統合）
============================================================
表記:
- 星の種類: ゲーム内カテゴリ名
- 見え方の核: 色以外で区別する現象
- 色: 代表的な色傾向（基本はパラメータ）
- 星の詳しい呼び名: UI/図鑑/自動命名に使う名称

[分類一覧]
1) 尾なし単色（Peony/牡丹系）
   - 核: その場で光って消える（尾ほぼ無し）
   - 色: 色だけ差し替え（赤/青/緑/黄/紫/橙/レモン/水色/ピンクなど）
   - 呼び名例: 紅星/青星/緑星/…（全部ここ）

2) 照明星（Illumination / Flare）
   - 核: 色より光度・燃焼時間が主目的（照明・信号）
   - 色: 白/黄が主（パラメータ化可）
   - 呼び名: 照明星（Flare）

3) 菊引き（Chrysanthemum “引”）
   - 核: 星が中心から尾を引いて放射状に伸びる
   - 色: 尾は炭火色/金系など（パラメータ扱いだが“尾タイプ”が主）
   - 呼び名: 「菊（引）」系、引先○○の“尾”側

4) 冠菊/柳引き（Kamuro/Willow “垂れ”）
   - 核: 尾が長く垂れ下がって落ちる（地上へ垂れる系）
   - 色: 金/銀/炭火など
   - 呼び名: 柳星、冠菊/錦冠菊/銀冠菊の“垂れ”

5) 和火尾（Wabi tail：渋い炭火尾）
   - 核: 明るさ控えめで“渋い”炭火尾（情緒系）
   - 色: 炭火色
   - 呼び名: 和火（Wabi）系（※一般的な炭火尾より暗く渋い表現）

6) 虎の尾（Tiger Tail：煙感が強い太尾）
   - 核: 極太で煤っぽい尾、煙感が強い
   - 色: 赤茶〜黒赤（表現上）
   - 呼び名: 虎の尾（Tiger Tail）

7) 錦/ブロケード金尾（Brocade：金の長尾）
   - 核: 明るい金色の長い尾（豪華）
   - 色: 金系
   - 呼び名: 錦星（Nishiki）／黄金星（短尾版もここ）／錦冠菊の金要素

8) 砂金（Gold dust：微細金尾）
   - 核: “サラサラ”した短く細かい金の尾
   - 色: 金系
   - 呼び名: 砂金星（Sakin）

9) 銀引（Silver line：鋭い銀尾）
   - 核: 銀色の鋭い直線・シャープ
   - 色: 銀系
   - 呼び名: 銀引（Gin-biki）

10) 銀冠（Silver Kamuro：銀の垂れ尾）
   - 核: “銀の柳”みたいに垂れる
   - 色: 銀系
   - 呼び名: 銀冠（Gin-kamuro）

11) 鉄火/鉄粉（Iron sparks：枝分かれ火花）
   - 核: オレンジ系火花が枝分かれする
   - 色: 金橙系
   - 呼び名: 鉄粉星/鉄火、松葉の“枝分かれ感”の核

12) グラナイト/スプレッダー（Granite/Spreader）
   - 核: ざらついた独特の火花＋煙っぽさ／広がり
   - 色: 青緑〜緑白が語られがち（パラメータ）
   - 呼び名: 苔/グラナイト（Granite）

13) グリッター（Glitter：遅れて瞬く粒）
   - 核: 遅れてキラキラ（時間遅れの瞬き）
   - 色: 金/銀が主
   - 呼び名: きらら（Kirara/Glitter）、金グリッター/銀グリッター

14) フリッター/スパンコール（Flitter/Spangle）
   - 核: 星そのものが毛羽立つ/金属片が鋭く光る（表面きらめき）
   - 色: 金/銀が主
   - 呼び名: 金フリッター/銀フリッター、スパンコール

15) ストロボ（Strobe：明滅）
   - 核: 一定周期でピカピカ明滅
   - 色: 白/赤/緑など（パラメータ）
   - 呼び名: 点滅星（Strobe）

16) クラックル（Crackle：微小破裂）
   - 核: パチパチ音＋閃光粒が飛ぶ
   - 色: 金銀っぽいことが多い（表現上）
   - 呼び名: 爆裂星（Crackle / Dragon Egg）

17) クロセット（Crossette：分裂）
   - 核: 燃焼途中で複数方向に割れて飛ぶ
   - 色: 色は載せ替え可
   - 呼び名: クロセット（Crossette）

18) 落葉（Falling leaves：ゆっくり落ちる）
   - 核: ゆらゆら・減速して落ちる挙動
   - 色: 色は載せ替え可
   - 呼び名: 落葉（Ochiba）

19) Afterglow（残光：青白い弱光）
   - 核: 消えそうで消えない青白い弱光（弱い残光）
   - 色: 青白系
   - 呼び名: 残光（Afterglow）／幽霊火（和名UIでは可）

20) 変色（Color change：層切替）
   - 核: 燃え進むと色が切り替わる（2〜数回）
   - 色: 色の順番が設計
   - 呼び名: 変色星、引先紅/引先青（“尾の種類”＋“先端の色層”）

4. 星の部品（層）ライブラリ：全モジュール一覧
============================================================
「必要な層」を選んで星（Star）に割り当てる。

表記:
- 部品（層）名: モジュール名（実装IDとしても使用可）
- 役割: 何を決める層か
- 対応分類: 上記「星の分類」のどの核に効くか
- 典型位置: スタック上の目安（時間順/優先順）

[構造・制御モジュール（見え方ほぼ無し）]
- Core / 芯（核）
  役割: 星の土台。寿命、基本サイズ、基準密度など。
  対応分類: すべて
  典型位置: Core

- Body coat / 充填・肉付け層
  役割: 星を太らせる“汎用肉付け”（エフェクト層の台）。
  対応分類: なし（構造）
  典型位置: Layer群の下支え（実装上はパラメータ）

- Adhesion coat / 接着・下地層
  役割: 次の層を安定して載せる（剥離防止）。ゲームでは「安定係数/発火率」を調整する枠。
  対応分類: なし（制御）
  典型位置: 層間

- Delay / 暗火（ディレイ）
  役割: 点火タイミングを遅らせる（見え方ほぼ無しで“間”を作る）。
  対応分類: 変色/合成の間
  典型位置: 中間

- Barrier / Blind（遮光・断熱・隔離）
  役割: 混色/暴走防止。ゲームでは「前段の残光を一旦落とす」「色にじみを抑える」。
  対応分類: 変色/複合
  典型位置: 層間

- Seal coat / 保護・シール層
  役割: 外装保護。ゲームでは「風耐性/湿気耐性」等のステータス枠。
  対応分類: なし
  典型位置: Prime外（メタ情報）

[起動モジュール（開始保証）]
- Prime / 化粧（通常プライム）
  役割: 着火性の安定・耐風性。ゲームでは開始の最小スパーク/フラッシュ。
  対応分類: すべて
  典型位置: Prime（最後に付与）

- Starter / Starting fire
  役割: 難起動スタックを確実に開始。ゲームでは「開始失敗率を下げる」。
  対応分類: 高負荷演出（ストロボ/クラックル等）
  典型位置: Prime寄り（任意）

- Dark prime / Changing relay（つなぎ）
  役割: 色変化の境界を“暗く”作る（混色区間を隠す）。ゲームでは切替の“黒味/暗転”。
  対応分類: 変色
  典型位置: 色層の間

[発光（色）モジュール]
- Steady Color（高輝度）色層
  役割: “点で一定発光”の色を作る。色はパラメータ。
  対応分類: 尾なし単色（牡丹系）
  典型位置: Layer1〜

- Steady Color（低輝度/渋色）色層
  役割: 暗め・深い色調の定常発光（雰囲気が違う）。
  対応分類: 尾なし渋色
  典型位置: Layer1〜

- Illumination / Flare層
  役割: 色より光度・持続（照明/信号）。
  対応分類: 照明星
  典型位置: Layer1〜

[尾（trail）モジュール]
- Chrysanthemum tail / 菊引き尾層
  役割: 放射状の“引き”尾。
  対応分類: 菊引き
  典型位置: 外側寄り（Layer1〜2）

- Willow/Kamuro tail / 柳・冠尾層
  役割: 長く垂れる尾。
  対応分類: 冠菊/柳
  典型位置: 外側寄り

- Wabi tail / 和火尾層
  役割: 渋く暗い炭火尾。
  対応分類: 和火尾
  典型位置: 外側寄り

- Tiger tail / 虎の尾層
  役割: 極太・煤っぽい尾（煙感が主役）。
  対応分類: 虎の尾
  典型位置: 外側寄り

- Brocade/Gold tail / 錦（金尾）層
  役割: 明るい金尾（豪華な長尾）。
  対応分類: 錦系
  典型位置: 外側寄り

- Gold dust / 砂金層
  役割: 微細でサラサラした短い金尾。
  対応分類: 砂金
  典型位置: 外側寄り

- Silver line / 銀引層
  役割: 鋭い銀色の直線尾。
  対応分類: 銀引
  典型位置: 外側寄り

- Silver kamuro / 銀冠層
  役割: 銀の“垂れ尾”（銀の柳）。
  対応分類: 銀冠
  典型位置: 外側寄り

[火花・きらめき・点滅・破裂モジュール]
- Iron sparks / 鉄火・鉄粉層
  役割: 枝分かれ火花（線香花火感）。
  対応分類: 鉄火/鉄粉
  典型位置: 外側寄り

- Granite/Spreader / 亜鉛グラナイト層
  役割: ざらついた独特の火花・広がり・煙っぽさ。
  対応分類: グラナイト
  典型位置: 外側寄り

- Glitter / グリッター層
  役割: 遅れてキラキラ（遅延フラッシュ）。
  対応分類: グリッター
  典型位置: 外側〜中間

- Tremolant / トレモラン層
  役割: グリッターのロング持続版。
  対応分類: グリッター（長持続）
  典型位置: 外側〜中間

- Flitter/Spangle / フリッター・スパンコール層
  役割: 金属片っぽい鋭い瞬き／毛羽立つ光。
  対応分類: フリッター/スパンコール
  典型位置: 外側

- Firefly overlay / ファイヤーフライ層
  役割: 尾の中に点がピカピカ混ざる（合成用オーバーレイ）。
  対応分類: 尾（T/M）＋微細きらめき
  典型位置: 外側

- Strobe / 点滅層
  役割: 周期的な明滅（暗期⇔閃光期）。
  対応分類: ストロボ
  典型位置: 外側〜中間

- Crackle / クラックル層
  役割: 微小破裂のパチパチ＋閃光粒。
  対応分類: クラックル
  典型位置: 内側〜外側

[挙動モジュール]
- Crossette split module / 分裂制御モジュール
  役割: 途中で分裂させる“割れ”設計（挙動）。
  対応分類: クロセット
  典型位置: 内部（効果層とセット）

- Falling Leaves drift module / 落葉挙動モジュール
  役割: ふわっと漂って落ちる（減速・揺らぎ）。
  対応分類: 落葉
  典型位置: 外側

[残光・終端モジュール]
- Afterglow / 残光層
  役割: 弱い青白・消えそうで消えない残光。
  対応分類: Afterglow（幽霊火）
  典型位置: 外側〜中間

- Terminal flash / 光露（終端フラッシュ）層
  役割: 消え際に“ピカッ”と立ち上げる終端属性。
  対応分類: 任意
  典型位置: 終端側

- Terminal slow-fade / 降雪（ゆっくり消える）層
  役割: 消え方を遅くする（綿雪っぽい）。
  対応分類: 任意
  典型位置: 終端側

5. 組み立てルール（データ設計）
============================================================
5.1 スタックの基本
- StarDef は以下の要素を持つ:
  - Core（必須）
  - Layers（0..N、推奨は最大6。拡張可）
  - Prime（原則必須）
  - Params（色・寿命・強度などのパラメータ）

- Layers は時間順（タイムライン）に評価される。
- “色”は原則 Params.color（または colorA/colorB/...）で差し替え。
- “尾の種類”がカテゴリ差（T/M系）を決め、色は尾に対するパラメータ。

5.2 合成ルール（推奨）
- 尾（T/M系） + Firefly overlay（任意） → 「尾の中で点が瞬く」合成が可能。
- Brocade（錦金尾） + Glitter/Flitter（任意） → 「豪華＋きらめき増し」。
- 変色は (ColorA -> Dark prime -> ColorB) を基本単位とし、段数分だけ繰り返す。

5.3 Prime の扱い
- 原則、全Starに Prime を付与（最小フラッシュ/開始スパーク）。
- 高負荷演出（ストロボ/クラックル等）は Starter を併用してもよい（開始安定用のパラメータ）。

```text
9. 星の特徴詳説（燃焼原理と時系列挙動）
============================================================
現実の煙火化学（Pyrotechnics）に基づく燃焼メカニズムと、物理法則（空気抵抗・重力・質量変化）に従った時間経過ごとの挙動詳細。

------------------------------------------------------------
A) ベース発光・変色系
------------------------------------------------------------

[1] 尾なし単色（Peony / 牡丹）
* 燃焼メカニズム: 「気相原子発光（Atomic Emission）」
    * 過塩素酸カリウム（酸化剤）と金属塩（着色剤）が反応し、高温のガス炎となって光る。星自体がガス化して消滅するため、燃えカス（スラグ）がほとんど残らない。
* 物理挙動:
    * 燃焼に伴い質量が急速にゼロに近づくため、運動エネルギーを失いやすい。
    * 弾道: 強い空気抵抗により、球状に広がった直後に「空中で静止」するような動きを見せる。
* 時間経過:
    * [T+0.00s] 開花: 猛烈な初速で放射状に飛び出す。色はまだ見えない（明るすぎて白に見える）。
    * [T+0.20s] 減速: 空気抵抗により急激にブレーキがかかる。ここで本来の色（赤/青など）が鮮明になる。
    * [T+0.50s] 静止・燃焼: ほぼ空中で静止した状態で、真球の形を保ちながら一定光度で燃える。重力による落下はほぼ視認できない。
    * [T+1.50s] 消滅: 燃え尽きると同時にスッと消える。残像は残らない。

[2] 尾なし渋色（Dim Peony / 渋星）
* 燃焼メカニズム: 「低温不完全燃焼」
    * 意図的に酸化剤を減らす、または炭酸塩を用いて燃焼温度を下げる。
* 物理挙動:
    * 燃焼が不安定であるため、光度曲線（Brightness Curve）が平坦ではなく、細かく振動（ゆらぎ）する。
* 時間経過:
    * [T+0.00s] 開花: 牡丹よりも初速にばらつきがあり、少し歪んだ円に広がる。
    * [T+0.50s] 滞留: 暗く鈍い光（パステルカラーや和色）で光る。背景の煙に少し埋もれる。
    * [T+1.20s] 消滅: 牡丹より早く、フワッと溶けるように消える。

[3] 照明星（Flare / Illumination）
* 燃焼メカニズム: 「金属酸化発光（Metal Combustion）」
    * マグネシウムやアルミニウム粉末が酸素と結合し、酸化マグネシウム（MgO）などの固体・液体微粒子となり、それが白熱して強烈な光（黒体放射＋原子発光）を出す。
* 物理挙動:
    * 高密度: 強くプレスされているため質量が大きい。
    * 慣性: 質量があるため空気抵抗に打ち勝ち、なかなか減速しない。
    * 重力: 減速後はその重さゆえに、はっきりと重力で落下する。
* 時間経過:
    * [T+0.00s] 開花: 非常に重いため、牡丹ほど遠くへ飛ばない。
    * [T+1.00s] 浮遊: 減速後、強い輝度を保ったまま滞空する。
    * [T+2.00s] 落下開始: 明確に放物線を描いて落ち始める。
    * [T+4.00s] 残存: 他の星が消えてもまだ光り続け、地面近くまで落ちていく。

[4] 変色 2段（Color Change 2-step）
* 燃焼メカニズム: 「積層燃焼（Layer Burning）」
    * 異なる組成が層になっており、外側から順に反応する。
* 物理挙動:
    * 色が変わる瞬間、燃焼ガスの噴出圧が変わるため、軌道がわずかに振動することがある（推力変化）。
* 時間経過:
    * [T+0.00s] 第1色: 色A（例: 青）で広がる。
    * [T+0.80s] 暗火（Dark）: 重要。中間層（Dark Prime）が燃える0.1〜0.2秒間、光がフッと消える（または極端に暗くなる）。これが無いと色が混ざる。
    * [T+1.00s] 第2色: 暗火を抜けた瞬間、色B（例: 赤）がパッと点灯する。
    * [T+1.50s] 消滅: 第2色が燃え尽きて消える。

[5] 変色 3段以上（Multi-step Change）
* 燃焼メカニズム: 「多層構造」
    * 2段と同様だが、層が薄くなるため製造精度（真球度）が挙動に直結する。
* 物理挙動:
    * 燃焼時間が長くなる傾向があり、後半は星が小さくなるため落下しにくくなる（空気抵抗の影響比率が増す）。
* 時間経過:
    * [T+0.0s] 色A: 発射。
    * [T+0.6s] 暗火→色B: 1回目の変化。
    * [T+1.2s] 暗火→色C: 2回目の変化。星の位置は開花点からかなり離れている。
    * [T+1.8s] 消滅: 最後は短めに終わることが多い。

[6] 引先○○（Tip Change / 尾＋先端変化）
* 燃焼メカニズム: 「芯（Core）の露出」
    * 外側の「尾を引く層」が燃え尽き、中心の「色を出す核」が露出した瞬間に性質が変わる。
* 物理挙動:
    * 尾を引いている間は「炭」の燃焼だが、先端の色が出るときは「ガス」の燃焼になるため、光の質感が「ザラザラ」から「ツルツル」へ変化する。
* 時間経過:
    * [T+0.00s] 尾の形成: 尾を引きながら飛行。
    * [T+1.00s] 変化点: 尾の供給が止まり、星本体だけが突然ピカッと色（例: 緑）を発する。
    * [T+1.20s] 点発光: その場で一瞬だけ色星として光り、すぐ消える。

[7] Afterglow（Afterglow / 残光）
* 燃焼メカニズム: 「燃焼生成ガスの励起」 または 「微細粒子の余熱」
    * 星が通過した後に残された微細なスラグやガスが、周囲の熱で淡く発光し続ける現象。
* 物理挙動:
    * 質量ゼロ: ガスや煙に近い状態。星の運動エネルギーを持たず、その場の気流（風）に流される。
* 時間経過:
    * [T+0.00s] 生成: 星が光っている間は見えない（明るさに負ける）。
    * [T+1.50s] 出現: 星本体が消滅した瞬間、その場所に「亡霊のような青白い光」が残っているのが見える。
    * [T+2.50s] 拡散: 形を失い、煙のように拡散して消える。

------------------------------------------------------------
B) 尾（Trail）系
------------------------------------------------------------

[8] 菊引き（Chrysanthemum）
* 燃焼メカニズム: 「炭素の熱放射（Black Body Radiation）」
    * 星に含まれる「木炭」の粒子が燃え残って後方に撒き散らされる。この炭粒子が空中で酸素と反応し、赤熱（Red Hot）状態で光る。
* 物理挙動:
    * 尾（炭）は非常に軽いため、放出された瞬間に空気抵抗でその場に止まる。星本体だけが前進するため、「線」が描かれる。
* 時間経過:
    * [T+0.00s] 開花: 星の移動経路に沿って、即座にオレンジ色の線が引かれる。
    * [T+0.50s] 伸長: 星が止まるまで尾が伸び続ける。
    * [T+1.00s] 尾の消失: 尾は「古い方（中心側）」から順に温度が下がり、赤→暗赤→黒と消えていく。先端（星側）が一番明るい。

[9] 冠菊/柳引き（Willow / Kamuro）
* 燃焼メカニズム: 「液状スラグの形成」
    * 大量の松炭とバインダーが燃焼時にお互いに溶け合い、粘り気のある「燃える液滴（スラグ）」となって星本体を覆う。
* 物理挙動:
    * 質量保存: ガス化せず液滴として残るため、星が軽くなりにくい。
    * 重力支配: 初速が落ちた後、重たい液滴の塊として重力落下を始める。
* 時間経過:
    * [T+0.00s] 開花: 菊同様に広がるが、少し初速が重い。
    * [T+0.80s] 停滞・垂れ始め: 空中で最高点に達した後、消えずにそのまま「下」へ向きを変える。
    * [T+2.00s] 落下: 金色の雨のように垂直に落ちる。光度は徐々に落ちる（金→暗赤）。
    * [T+3.00s] 接地: 地面に届くほど長く残る場合がある。

[10] 和火尾（Wabi / 炭火）
* 燃焼メカニズム: 「低温炭素燃焼」
    * 黒色火薬に近い組成。燃焼温度が低く、CO2の発生圧力が弱いため、推進力が安定しない。
* 物理挙動:
    * 星が不規則に揺らぐ（Wobble）。
    * 尾の粒子密度が低く、パラパラと散らかる。
* 時間経過:
    * [T+0.00s] 開花: 線が細く、頼りない感じで広がる。
    * [T+0.50s] ゆらぎ: まっすぐ飛ばず、少しよろけるような軌道を描く。
    * [T+1.00s] 消失: 燃え尽きるのが早い。「儚さ」の表現。

[11] 虎の尾（Tiger Tail）
* 燃焼メカニズム: 「大量固形分噴出」
    * 炭素と共に、燃焼しても気化しない「固形灰分」を大量に出す組成。
* 物理挙動:
    * 体積: 光ではなく「物質」の柱であるため、太さとボリューム感がある。
    * 乱流: 尾の内部で煙が渦を巻く（Turbulence）。
* 時間経過:
    * [T+0.00s] 噴出: ドカンと太い柱が伸びる。先端よりも「根本」が太く見えることもある。
    * [T+1.00s] 滞留: 星が止まった後も、煙の柱がその場に長く残る。
    * [T+1.50s] 冷却: 光が消えた後もしばらく「黒い煙」として視認できる。

[12] 錦/ブロケード（Brocade）
* 燃焼メカニズム: 「チタン/フェロチタン燃焼」
    * 炭（菊）の中に金属粉を混ぜることで、炭の赤熱（暗い）＋金属の白熱（明るい）が混ざり、豪華な金色になる。
* 物理挙動:
    * 金属粉は炭より重く、慣性があるため、尾の幅が少し広がりやすい。
* 時間経過:
    * [T+0.00s] 開花: 菊より圧倒的に明るい金色の線。
    * [T+1.00s] 垂れ: 柳ほどではないが、先端が少し重力で垂れ下がる（しだれ柳）。
    * [T+2.00s] きらめき: 尾の中に混ざった金属粒がチカチカと最後まで光る。

[13] 砂金（Gold Dust）
* 燃焼メカニズム: 「微細金属の瞬時燃焼」
    * 星から放出された微細な金属粉が、空気中で即座に酸化反応を終える。
* 物理挙動:
    * 粒子が極めて軽いため、放出された瞬間に静止し、瞬時に燃え尽きる。線にならず「点」の集合に見える。
* 時間経過:
    * [T+0.00s] 放出: 星の後ろに「粉」を撒く。
    * [T+0.10s] 明滅: 撒かれた粉が「サラサラ」と一瞬で消える。
    * 連続性: これが連続することで、天の川のような帯になる。

[14] 銀引（Silver Line）
* 燃焼メカニズム: 「アルミニウム粉末燃焼」
    * アルミニウムの燃焼光は純白で鋭い。
* 物理挙動:
    * 燃焼ガス圧が高く、星が推進力を持つため、軌道が直線的（直進性が高い）になる傾向がある。
* 時間経過:
    * [T+0.00s] 射出: 鋭いレーザーのような光線が走る。
    * [T+0.50s] 停止: 先端が鋭く止まる。
    * [T+0.60s] 消失: 尾の持続時間は短く、パッと消える。

[15] 銀冠（Silver Kamuro）
* 燃焼メカニズム: 「アルミニウム・スラグ」
    * 錦のアルミニウム版。アルミ粉末を多量に含んだスラグを形成する。
* 物理挙動:
    * アルミニウムは炭素より軽いため、金の柳（炭素ベース）よりも落下速度が遅く、ふんわりと垂れる。
* 時間経過:
    * [T+0.00s] 開花: 白銀の線が広がる。
    * [T+1.00s] 垂れ: 優雅に垂れ下がる。金よりも「軽い」印象で落ちる。

------------------------------------------------------------
C) 火花・きらめき・挙動系
------------------------------------------------------------

[16] 鉄火/鉄粉（Iron Sparks）
* 燃焼メカニズム: 「炭素拡散爆発（Carbon Diffusion）」
    * 放出された鉄粉が高温で溶け、その内部に炭素が固溶する。冷却過程や酸化過程で内部からCO2ガスが発生し、内圧に耐え切れなくなった鉄液滴が物理的に破裂（Burst）し、枝分かれする。
* 物理挙動:
    * 二段階飛翔: [親星] → [放出された鉄粒] → [破裂して飛ぶ子粒]。
* 時間経過:
    * [T+0.00s] 飛行: 星が飛ぶ。尾はあまり目立たない。
    * [T+0.20s] 分岐: 星の後方で、放出された粒が「パチッ」と音もなく枝分かれする。
    * [T+0.50s] 連鎖: 枝分かれした先でさらに細かく分かれ、線香花火のような形状を作る。

[17] グラナイト（Granite）
* 燃焼メカニズム: 「亜鉛の昇華燃焼」
    * 亜鉛（Zinc）などは沸点が低く、燃焼時に激しく沸騰・揮発し、酸化亜鉛の微粒子（煙）を伴いながら細かく砕ける。
* 物理挙動:
    * 振動: 激しいガス噴出により、星自体がジリジリと微振動する。
    * 拡散: 輪郭がボヤけた、毛羽立ったような火花になる。
* 時間経過:
    * [T+0.0s]〜[End]: 終始「ジリジリ」という音（Fry sound）と共に、ミミズが這うような不安定な軌跡と、モヤのような火花を撒き散らす。

[18] グリッター（Glitter / きらら）
* 燃焼メカニズム: 「スプリットレル反応（Spritrel Reaction）」
    * マグナリウム合金と硫化物の反応により、星表面に「反応性の高い溶融液滴（ドロップ）」が生成される。この液滴は星から離れた後、空中の酸素と反応して遅れて「爆発（Flash）」する。
* 物理挙動:
    * 分離: 星本体から液滴が物理的に分離する。
    * 遅延: 分離した液滴は、化学反応が進むまでの数ミリ秒〜数百ミリ秒間、光らずに空間を漂う（暗反応）。
* 時間経過:
    * [T+0.00s] 通過: 星が通過。あまり光らない。
    * [T+0.30s] 発光: 星が通り過ぎた「何もない空間」で、置き去りにされた液滴がパパパッとカメラのフラッシュのように光る。
    * [T+0.40s] 消失: 光った瞬間に消滅する。

[19] フリッター/スパンコール（Flitter）
* 燃焼メカニズム: 「金属片の物理反射と燃焼」
    * 粉末ではなく「箔（Flake）」状のアルミニウムなどを使用。
* 物理挙動:
    * 回転: 平たい金属片が空気抵抗でクルクルと回転（Tumbling）しながら飛ぶ。
    * 異方性: 金属片の面がこちらを向いた時だけ強く光る。
* 時間経過:
    * [T+0.00s]〜[End]: 連続的な線ではなく、パラパラとした断続的な強い煌めきとして見える。

[20] ストロボ（Strobe / 点滅）
* 燃焼メカニズム: 「振動燃焼（Oscillatory Combustion）」
    * 暗期（Smolder）: 低温（約400℃）でゆっくり反応し、熱を蓄積する。光はほぼ出ない。
    * 明期（Flash）: 蓄積した熱が臨界点を超えると、高温（約2000℃）の反応が一気に連鎖し、激しく発光する。反応物が尽きるとまた暗期に戻る。
* 物理挙動:
    * 明期の瞬間にガス圧（推力）が発生するため、星がピクッピクッと動く。
* 時間経過:
    * 周期: 3Hz〜10Hz（1秒に3〜10回）。
    * [T] 暗（0.1s）: 見えない。
    * [T+0.1s] 明（0.05s）: ピカッと光る。
    * [T+0.15s] 暗: また消える。これを繰り返す。

[21] クラックル（Crackle / 爆裂）
* 燃焼メカニズム: 「爆轟（Detonation）類似反応」
    * 酸化ビスマスや四酸化三鉛（鉛丹）を核とした「マイクロスター（Dragon Egg）」が、熱で還元され、一気に爆発的反応を起こす。
* 物理挙動:
    * 燃焼ではなく「爆発・破砕」。星が粉々になって吹き飛ぶ。
* 時間経過:
    * [T+0.00s] 潜伏: 通常の星として飛ぶ（または暗い状態で飛ぶ）。
    * [T+1.00s] 破裂: 寿命が来た瞬間、「パチパチパチ！」という音と共に、半径数十センチの範囲に激しい閃光火花を撒き散らして消滅する。

[22] クロセット（Crossette / 分裂）
* 燃焼メカニズム: 「内圧破断」
    * 星の中に仕込まれた「割薬（Burst Charge）」に火が回ると、内部からガスが発生し、外殻を物理的にへし折る（Split）。
* 物理挙動:
    * 作用反作用: 分裂した破片は、爆発の勢いで加速（Boost）され、元の軌道から逸れて飛んでいく。
* 時間経過:
    * [T+0.00s] 親星飛行: 1つの星として飛ぶ。
    * [T+0.50s] 分裂: 空中でパンッ！と割れ、4つ（十字）または複数の破片になる。
    * [T+0.55s] 拡散: 破片がそれぞれ別の方向へ、加速しながら飛ぶ（網目状になる）。

[23] 落葉（Falling Leaves）
* 燃焼メカニズム: 「層流燃焼と空力設計」
    * ガス発生剤を多く含み、星自体が軽く平たい形状をしていることが多い。
* 物理挙動:
    * 揚力（Lift）: 不規則な形状やガス噴出により、翼のような揚力が発生する。
    * 空気抵抗: 非常に大きく、重力落下速度が極端に遅い。
* 時間経過:
    * [T+0.00s] 放出: 爆発後、すぐに減速して止まる。
    * [T+0.50s] 漂流: 左右にゆらゆらとスライド（Swaying）しながら、ゆっくりと落ちていく。
    * [T+3.00s] 明滅: 種類によっては、落下しながら色が切り替わる。

============================================================
10. 層（Layer）の役割と化学的挙動詳細：全モジュール学術的解説
============================================================
本章では、各モジュールにおける化学反応式、熱力学挙動、および物理的相転移（Phase Transition）を詳細に定義し、それに紐づくゲーム内での視覚表現（Rendering & Visuals）を規定する。

[構造・制御モジュール（Structure & Control）]
1. Core / 芯（核）
物理実体詳細:
伝統的素材: 菜種（Rape seed）、ケシの実、またはそれらに微量の黒色火薬をコーティングしたもの。
現代的素材: セラミックボール、高密度プラスチック、またはカットされた星の破片（Cut star）。
物理・弾道学的挙動:
弾道係数（Ballistic Coefficient）: 芯の密度が星全体の最終的な密度を左右する。植物種子は比重が軽いため空気抵抗で減速しやすいが、セラミック芯（比重2.0以上）は慣性モーメントが大きく、真円度が高いため、より遠くまで飛翔し、開花半径（Bloom Radius）を物理的に拡大させる効果がある。
吸湿バッファ: 植物性芯は微量の水分を吸放出するバッファとして機能し、火薬層の乾燥割れを防ぐが、製造精度は落ちる。
燃焼後の残留:
通常は燃焼せずに炭化して落下するが、ゲーム描画上は無視できる（サイズ0.5mm以下）。
【視覚表現 (Visuals)】:
    * 不可視: 基本的に描画しない（透明）。
    * ピボット: 全てのパーティクルの「発生中心点」として機能する。
    * 消滅時: 燃焼終了時に、ごく稀に0.1秒未満の小さな「暗い赤点（残り火）」として描画してもよいが、基本は無視。

2. Body coat / 充填・肉付け層
化学組成:
硝酸カリウム（$KNO_3$） + 炭素（C） + 大量のデキストリンまたは木粉。
熱力学的挙動:
低カロリー燃焼: 酸化剤の比率を化学量論的（Stoichiometric）な完全燃焼比率より意図的に下げてある。これにより燃焼温度を抑え、発光を抑制する。
断熱膨張: 燃焼時に発生するガス（$CO_2, N_2$）が緩衝材となり、星のサイズを物理的に大きくする（直径調整）ためだけに使われる。
実装パラメータへの影響:
この層の厚みが増すほど、星の「寿命（Lifetime）」は変わらずに「初期サイズ（Start Size）」だけが大きくなる。
【視覚表現 (Visuals)】:
    * 暗黒燃焼: 光らない。
    * サイズ定義: 星の発光直径（Point Size）の「初期値」を決定する。層が厚いほど、点火直後の星のサイズが大きくなる。

3. Adhesion coat / 接着・下地層
化学的機能:
界面結合（Interfacial Bonding）: 異なる熱膨張係数を持つ層（例: 高温で膨張する金属層と、脆い炭層）の間に介在し、物理的なストレスを緩和する。
組成: バインダー（可溶性澱粉や合成樹脂）の濃度を通常5%から10〜15%に高めたスラリー状の火薬。
燃焼伝播メカニズム:
この層は「燃え移り（Pass-fire）」の成功率を支配する。表面エネルギーが高くなるように設計されており、上層の燃焼熱を受け止め、下層へ効率よく熱伝導（Thermal conduction）させる。ここが欠落すると、燃焼が層の境界で止まる「中抜け現象」が発生する。
【視覚表現 (Visuals)】:
    * アンチエイリアス: 異なる色層の間に、1フレーム以下の極短時間の「混色」または「高輝度スパイク」を発生させ、色の切り替わりを滑らかにする（唐突な切り替わりを防ぐ）。

4. Delay / 暗火（ディレイ）
化学組成と反応:
組成: 硝酸カリウム（$KNO_3$） + 桐炭（Paulownia charcoal） + 硫黄（S） + 珪藻土（増量剤）。
反応式: $2KNO_3 + S + 3C \rightarrow K_2S + N_2 + 3CO_2$ （黒色火薬反応だが、炭素過多で反応速度を遅らせてある）。
熱力学的挙動:
Solid-Phase Reaction: 気相での炎を出さず、固相表面での赤熱反応（Smoldering）として進行する。
温度制御: 燃焼温度は約800℃〜1000℃と低く、可視光放射（黒体放射）が極めて弱い。
視覚的効果の正体:
夜空（暗黒）に対して輝度が閾値以下であるため、「消えている」と認識される。実際には赤外線領域で弱く発熱している。
【視覚表現 (Visuals)】:
    * 完全透明: アルファ値 = 0。星が「消えている」状態。
    * 赤外線: 暗視スコープエフェクト等でない限り、可視光はゼロ。
    * 軌跡の断絶: グリッター等の場合、星の位置と発光位置の間に「何も描画されない空間」を作る。

5. Barrier / Blind（遮光・断熱・隔離）
化学的必然性:
酸化還元分離: 例えば、塩素酸カリウム（$KClO_3$）を含む色層の上に、硫黄（S）を含む層を直接乗せると、経年劣化で酸（$H_2SO_4$）が発生し、接触面で自然発火する危険がある。これを防ぐための化学的に不活性な壁。
視覚的機能:
光のシャッター: 前の層の残光（Afterglow）が完全に消えるまでの時間（数十ミリ秒）を稼ぐ。これにより、人間の網膜上の残像がリセットされ、次の色が鮮やかに知覚される。
【視覚表現 (Visuals)】:
    * ブラックアウト（光のシャッター）: 色変化の際、前の色のトレイル（残像）が完全にフェードアウトし、画面上が「黒」になってから次の色が点灯する。色のメリハリ（Contrast）を生む重要な「間」。

6. Seal coat / 保護・シール層
材料工学:
素材: ニトロセルロース（NC）ラッカー、シェラック、またはクラフト紙の積層。
物理的挙動:
耐衝撃性: 発射時（Launch）、筒底では数百〜数千Gの衝撃と、数千度・数十気圧の発射ガスに晒される。この層が無いと、星が粉砕されるか、表面のPrime層が吹き飛ばされて点火しなくなる。
防湿: 過塩素酸塩や硝酸塩は吸湿性が高く、水分を含むと点火しなくなるため、これを遮断する疎水性バリア。
【視覚表現 (Visuals)】:
    * マズルフラッシュ: 発射（Launch）の瞬間、筒口で一瞬だけ「白っぽい煙」として気化するが、上空での開花時には存在しない。

[起動モジュール（Ignition）]
7. Prime / 化粧（通常プライム）
化学組成:
過塩素酸カリウム（70%） + 木炭 + 赤松脂（Red gum） + アルミニウム粉末（微量）。これを「まぶし」として最外層に形成。
熱力学メカニズム（Step-up Ignition）:
活性化エネルギーの壁: 星の内部（特に金属星や青色星）は、点火に必要な活性化エネルギーが非常に高い（火がつきにくい）。
スラグ熱伝導: Primeに含まれるシリコンや金属粉が燃焼して「高温の液体スラグ」となり、星の表面に物理的に付着する。
プロセス: [割薬の爆風] $\rightarrow$ [Prime着火(約400℃)] $\rightarrow$ [Prime燃焼(約2000℃+スラグ形成)] $\rightarrow$ [星本体への熱侵徹] $\rightarrow$ [星本体の自己分解開始]。
このプロセスは数ミリ秒で行われる。
【視覚表現 (Visuals)】:
    * アタック（Attack）: 開花した瞬間、全星が一斉に「金白色」の高輝度フラッシュ（Bloom強度：高）を放つ。
    * 持続: 0.05秒〜0.1秒程度。
    * ハレーション: 星の本来の色が見える前に、一瞬だけ視界を白く染める演出。

8. Starter / Starting fire
特殊用途:
マグネシウム含有量の高いストロボ星や、着火温度が高い有機系色星に用いる。
化学挙動:
テルミット類似反応: 酸化鉄（$Fe_3O_4$）とアルミニウム（$Al$）の反応のような、ガス発生が少なく高熱のみを発生する反応を利用し、星の表面を「焼き切る」ようにして強制着火させる。
【視覚表現 (Visuals)】:
    * 予熱光: Primeのフラッシュが終わった後、本来の色が出る前に一瞬挟まる「オレンジ色」または「白熱色」の短い区間。

9. Dark prime / Changing relay（つなぎ）
光学的機能:
明順応のリセット: 明るい星（Flare）から暗い星（牡丹）へ変化する際、中間に「暗黒」を挟むことで、人間の目の感度（絞り）を開かせ、次の暗い色をより鮮やかに見せる効果（心理物理学的効果）。
燃焼速度整合:
色層と同じ燃焼速度（Burning rate）を持つように調整されているため、星の直径変化（$d(t)$）のグラフを歪ませない。
【視覚表現 (Visuals)】:
    * ブリンク（Blink）: 色Aと色Bの間で、星が一瞬「フッ」と消える。
    * 残像切断: 軌跡（Trail）が物理的に途切れる。

[発光（色）モジュール（Color Emission）]
10. Steady Color（高輝度）色層
発光原理: 分子発光（Molecular Emission）
原子そのものではなく、燃焼中の高温化で生成される「不安定な一塩化物（Monochloride）」が特定の波長を出す。
赤: 塩化ストロンチウム（$SrCl$）。波長 600-640nm。
緑: 塩化バリウム（$BaCl$）。波長 500-530nm。
青: 塩化銅（$CuCl$）。波長 420-450nm。
化学反応詳細（青色星の例）:
酸化剤（$KClO_4$）が酸素を供給。
塩素供与剤（Parlon/PVC）が分解して塩素（$Cl$）と燃料を供給。
酸化銅（$CuO$）が反応。
反応式: $CuO + HCl(from PVC) \rightarrow CuCl(\text{発光種}) + OH$。
重要: 温度が高すぎると（>2000℃）、$CuCl$が分解して$Cu$原子や$CuOH$になり、青色が失われる（White-out）。そのため、燃焼温度の厳密な制御が必要。
【視覚表現 (Visuals)】:
    * テクスチャ: 輪郭がはっきりした「ハードエッジ」な円形。
    * 透明度: 不透明に近いガス球。
    * ブルーム: 色そのものの彩度が高く、中心部は白く飛ばずに「濃い色」のまま光る（例：芯まで青い）。
    * 煙: ほとんど出ない（スモークレス）。

11. Steady Color（低輝度/渋色）色層
発光原理:
塩素供与剤（Parlon等）を使用せず、酸化物（$SrO$や$BaO$）の発光を利用する古い配合。
分光特性:
$SrCl$（深い赤）に対し、$SrO$はオレンジがかった朱色になる。
スペクトルがブロード（広帯域）になり、彩度が落ちるが、これが「和火」や「パステル」の風合いとなる。
【視覚表現 (Visuals)】:
    * テクスチャ: 輪郭がぼやけた「ソフトパーティクル」。
    * 色調: パステルカラー、または和色（柿色、草色）。彩度は低め。
    * ゆらぎ: 輝度（Intensity）が一定ではなく、微妙にチラチラと明滅するノイズが乗る。

12. Illumination / Flare層
発光原理: 黒体放射（Black Body Radiation） + 輝線スペクトル
化学反応:
マグネシウム（$Mg$）の燃焼: $2Mg + O_2 \rightarrow 2MgO + \text{Heat}$。
反応熱は3000℃近くに達する。
物理挙動:
この温度では、生成された酸化マグネシウム（$MgO$）粒子が白熱し、プランクの法則に従った連続スペクトルの強力な白色光を出す。さらにナトリウムなどが含まれると、原子吸光による強烈な輝線が加わる。
ブルーミング: 輝度が高すぎてカメラや人間の目の中で散乱を起こし、光源サイズよりも大きく見える。
【視覚表現 (Visuals)】:
    * グレア（Glare）: 画面全体に光が溢れ出すような強いレンズフレア効果。
    * 色: 純白、またはわずかに青みがかった白。
    * ハロー: 星の周囲に大きな光輪が発生する。
    * 残像: 網膜に焼き付くような、長い減衰（Long decay）を持つ。

[尾（Trail）モジュール詳細]
13. Chrysanthemum tail / 菊引き尾層
現象: After-burning effect
プロセス:
星内部で、酸化剤に対して燃料（木炭）が過剰な状態で燃焼する。
未燃焼の炭素粒子（C）が熱せられた状態で星から放出される。
放たれた炭素粒子が空気中の酸素を取り込み、表面燃焼を行う。
温度は約1000℃〜1200℃。ウィーンの変位則により、この温度帯は「赤〜オレンジ色」の放射となる。
物理:
炭素粒子は多孔質（Porous）であるため、比表面積が大きく、空気抵抗を受けやすい。そのため放出直後に減速し、綺麗な尾を形成する。
【視覚表現 (Visuals)】:
    * 形状: 星から伸びる、隙間のない一本の線（Line Strip）。

14. Willow/Kamuro tail / 柳・冠尾層

15. Wabi tail / 和火尾層

16. Tiger tail / 虎の尾層

21. Iron sparks / 鉄火・鉄粉層
現象: Branching Sparks (Senko-Hanabi phenomenon)
メカニズム:
星から放出された鉄粉（Fe）が酸化し、液状の酸化鉄（FeO/Fe2O3）の球になる。
液滴内部に、未反応の炭素（C）が溶け込む。
炭素が酸化鉄中の酸素と反応し、一酸化炭素（CO）ガスを生成する。
液滴内部のガス圧が高まり、表面張力を超えた瞬間に破裂（Burst）する。
破裂した微細な液滴が再び同じプロセスを経て、二次、三次の破裂（松葉状分岐）を起こす。
【視覚表現 (Visuals)】:

    * 軌道: 分裂した粒子は、速度ベクトルがランダムに変化し、ジグザグに飛ぶ。
    * 色: 高温の橙色から、分裂するたびに赤く暗くなる。

22. Granite/Spreader / 亜鉛グラナイト層

反応:
亜鉛（Zn）の融点は419℃、沸点は907℃と、火薬の燃焼温度に比べて極端に低い。
このため、燃焼中に亜鉛は「燃える」というより「激しく沸騰・蒸発」する。
蒸発した亜鉛ガスが空気中で酸化亜鉛（ZnO）の煙となり、緑色がかった白い炎と、ジリジリという沸騰音（Audible noise）を生む。

    * 色: 緑色がかった白、またはクリーム色。

23. Glitter / グリッター層
反応名: Spritrel Reaction (Sputter)
化学メカニズム:
成分: 硝酸バリウム/ストロンチウム + アルミニウム + 硫黄 + 硫化アンチモン。
Phase 1 (Dross Formation): 燃焼中、星の表面にアルミニウムと硫化物の溶融混合物（Dross droplet）が生成される。
Phase 2 (Delay): この液滴は星から離脱するが、すぐには光らない（化学的誘導期）。
Phase 3 (Flash): 液滴中の硫化アルミニウム（$Al_2S_3$）が空気中の酸素と急激に反応し、爆発的に燃焼する。
視覚:
この「誘導期（Delay）」が存在するため、星本体から数センチ〜数十センチ離れた空間で発光現象が起きる。
【視覚表現 (Visuals)】:
    * 空間発光: 星本体（親）の位置ではなく、星が通り過ぎた「虚空」で光が生まれる。
    * 点滅パターン: 正弦波（Sine）ではなく、スパイク状（Sawtooth）の急激な明滅。パッと光ってすぐ消える。
    * 色むら: 化学反応の不均一さにより、金・白・黄色がランダムに混ざる。

24. Tremolant / トレモラン層
変種: Long-delay Glitter
化学調整:
グリッター組成に対し、木炭や重炭酸ナトリウムを添加して反応を抑制し、ドロスのサイズを大きくする。
大きなドロスは熱容量が大きいため冷めにくく、また空気抵抗に対する慣性も大きいため、長く揺らぎながら垂れる挙動になる。
【視覚表現 (Visuals)】:
    * 揺らぎ: 光る粒が消えずに、空中でユラユラと漂いながら少し垂れる。
    * 持続: グリッターよりも「粘り」のある光。

25. Flitter/Spangle / フリッター・スパンコール層
物理光学: Specular Reflection
素材:
アトマイズ（球形）ではなく、スタンプミルで叩いて延ばした「フレーク（鱗片）状」のアルミニウム。
挙動:

燃焼光に加え、周囲の光を鏡面反射するため、連続光ではなく「チラチラ」とした金属光沢感が出る。

26. Firefly overlay / ファイヤーフライ層
構造: Matrix Inclusion
製造法:
あらかじめ作成した小さな金属星（Micro-stars, 1mm程度）を、湿らせた菊星の組成（Matrix）に混ぜ込んでプレスする。
燃焼:
ベースの菊（炭）が一定速度で後退（燃焼）していく。
燃焼面がマイクロスターに到達すると、それがポロリと外れ、独立して点火・発光する。
ランダム分布しているため、不規則なタイミングで光る。

27. Strobe / 点滅層
反応分類: Oscillatory Combustion (振動燃焼)
化学サイクル:
Dark Phase (Smolder): 酸化剤（硝酸バリウム等）とマグナリウムが約400℃〜500℃で反応。マグネシウム主体で、発光は弱く、主に熱を蓄積し、中間生成物（溶融塩）を作る。
Flash Phase (Explosion): 蓄積温度が臨界点（約1000℃以上）を超えると、中間生成物が一気に分解反応を起こし、2000℃以上の高温スパイクとガス発生を引き起こす。
Reset: 燃料と酸化剤が局所的に消費され、ガス膨張で冷却されるため、再びDark Phaseに戻る。
周波数制御:
マグナリウムの粒子サイズや、硫酸塩の比率で明滅周波数（Frequency）が決まる。
【視覚表現 (Visuals)】:
    * 矩形波（Square Wave）: オンとオフがデジタル的に切り替わる。フェードイン・アウトがない。
    * 推力振動: 光る瞬間に、星がピクッと少し前に進む（微細な加速）。
    * ハレーション: オンの瞬間だけ、強いブルームが発生する。

28. Crackle / クラックル層
反応名: Lead/Bismuth Oxide Reaction (Dragon Eggs)
組成:
四酸化三鉛（$Pb_3O_4$）または酸化ビスマス（$Bi_2O_3$） + マグナリウム（$MgAl$） + 酸化銅（$CuO$）。
反応メカニズム:
通常、ニトロセルロースで固められたコアとして存在する。
点火されると、最初は静かに燃えるが、内部で金属酸化物がマグナリウムによって還元される発熱反応が進む。
金属が溶融状態（Liquid phase）になり、反応速度が指数関数的に加速（Thermal runaway）。
一瞬で沸騰・ガス化し、**超音速の衝撃波（Sonic boom）**を伴う微細爆発を起こす。これが「パチッ」という音の正体。
【視覚表現 (Visuals)】:
    * バースト: 星が粉砕され、半径50cm〜1mの範囲に瞬間的に火花が広がる。
    * 閃光: カメラのストロボのような、フレーム単位の真っ白な閃光。
    * 残像なし: 一瞬で事象が終了する。

30. Falling Leaves drift module / 落葉挙動モジュール
流体力学: Aerodynamic Instability
設計:

形状を球（Sphere）ではなく、扁平なディスク状または角柱状に成形する。

31. Afterglow / 残光層
発光原理: Incandescence of Ash & Chemiluminescence
化学:
燃焼の主反応が終了した後も、微細なスラグ粒子（Ash）が熱を持ち続け、赤外線〜暗赤色の光を出し続ける。
または、特定のガス分子（NO2等）が冷える過程で化学発光（Chemiluminescence）を起こす。
環境依存:

【視覚表現 (Visuals)】:

    * ブレンド: 加算合成（Additive）により、夜空をほんのり明るくする。

32. Terminal flash / 光露（終端フラッシュ）層
熱力学: Geometric Thermal Concentration
原理:
球体の星が燃焼して小さくなると、体積に対する表面積の比率は変わるが、熱が逃げる距離（半径）が短くなる。
中心部（Core付近）は周囲からの燃焼熱で予熱（Pre-heating）され続けており、温度が最高潮に達している。
そこに、反応性の高い組成（High-energy composition）を配置することで、熱暴走的な急激な燃焼（Flash）を引き起こす。
【視覚表現 (Visuals)】:
    * サイズアップ: 消える直前の1〜2フレームだけ、星のスケールが1.5倍〜2倍に膨らむ。
    * ホワイトアウト: 色が飛び、純白になってから消滅する。

33. Terminal slow-fade / 降雪（ゆっくり消える）層
化学制御: Stoichiometry Gradient
原理:
星の中心に向かうにつれて、酸化剤の比率を徐々に下げていく（Fuel-richにする）。
酸素供給が減るため、反応速度（Reaction rate）と温度が徐々に低下する。
これにより、光度が急にゼロになるのではなく、アナログ的な減衰曲線（Decay curve）を描いて消滅する。
【視覚表現 (Visuals)】:
    * イージング: 輝度カーブが「Log的」に減衰する。リニアに消えるよりも「粘り」を感じさせる。
    * 赤方偏移: 消え際にかけて、色が少し赤み（長波長）に寄っていく（温度低下の表現）。

```
